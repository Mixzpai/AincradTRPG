# AincradTRPG

A text-based RPG game inspired by the world of Sword Art Online's Aincrad. Built with .NET 8, this console application lets players create characters, battle monsters, manage inventory, and progress through a stat-based combat system.

## Features

- **Character Creation** – Create your own adventurer with custom name and attributes
- **Stat System** – Manage core stats including Vitality, Strength, Endurance, Dexterity, Agility, and Intelligence
- **Combat System** – Battle monsters and boss encounters with damage calculations based on player and enemy stats
- **Inventory Management** – Equip weapons, armor, and accessories; use consumables like potions and food
- **Equipment Bonuses** – Gear provides stat modifiers that enhance your character's abilities
- **Experience & Leveling** – Gain experience from defeating monsters, with overkill damage bonuses
- **Item Rarity System** – Items range from Common to Rare with varying stat effects
